# GitPeek Documentation

Welcome to the GitPeek documentation! GitPeek is a modern, Spotify-inspired web application for exploring GitHub profiles, repositories, and user statistics.

Our documentation is structured following the **Diátaxis** framework, meaning it is divided into four distinct parts to serve different user needs at different times.

## 🧭 Navigation

- **[Tutorials](./tutorials/)**: Step-by-step guides for learning how to use and deploy GitPeek from scratch. Start here if you are new to the project.
- **[How-to Guides](./how-to/)**: Goal-oriented guides that show you how to solve specific problems or perform specific tasks (e.g., customizing the design, searching effectively).
- **[Explanation](./explanation/)**: Conceptual discussions that clarify how GitPeek works under the hood, including its architecture and core features.
- **[Reference](./reference/)**: Technical descriptions of the machinery and how to operate it, such as API endpoints and the technology stack.
